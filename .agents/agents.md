# MemTrace Antigravity Agents

Antigravity should treat `agent.md` as the authoritative project operating
guide for this repository.

## Shared Constraint

Every Antigravity persona or workflow in this workspace must read and follow
`agent.md` before non-trivial planning, coding, review, verification, or
specification work.

## Required Behavior

- Use the private Agent Loop KB for workflow, gate, handoff, and task-state
  rules.
- Write discussion, planning, development, and acceptance outcomes back to
  `ws_spec_plan`.
- After final development that changes public behavior or contracts, update
  the public Chinese/English specs and seed artifacts named in `agent.md`.
- Keep this file as a thin Antigravity entry point. Do not copy the full policy
  here.
