# MemTrace Agent Entry Point

This repository's shared agent operating guide is `agent.md`.

Before non-trivial planning, coding, review, or specification work, read and
follow `agent.md`. It defines the required Agent Loop workflow, private
MemTrace KB lookup, `ws_spec_plan` write-back, and public spec/seed
synchronization rules.

Tool-specific wrappers may point here or directly to `agent.md`; do not copy
the full policy into multiple files.
